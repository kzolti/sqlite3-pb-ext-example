# sqlite3_pb_ext_example
example for sqlite3_pb_ext_gen project

Init repo
```
git clone git@github.com:kzolti/sqlite3_pb_ext_example.git \
&& cd sqlite3_pb_ext_example 
```

Run cmake and build 
```
mkdir -p build && cd build \
&& cmake .. && cmake --build . \
```

Run example:
```
./sqlite_pb_ext_example
```
